Budget Header

The budget screens all include a header with information about the Proposal and the Budget Version.

 

Left –

Description

Proposal number & Budget Version number

The proposal number.

The open budget version. Many versions can be added, but only the one marked “complete” and “for submission’ is routed into workflow for review, approval, and submission.

Budget #: Name

The open version number of the budget and the name entered by the user when the budget was created (the name is only used for reference by the user).

Right –

Description

Final Version (Yes/No)

“Yes” means this budget version will be used for submission.

Created (date)

The date this budget version was created.

Parent Proposal

The proposal number that this budget is a part

‘more’

Click to display information expanded including the PI, created and last updated timestamps as well as the sponsor name.
