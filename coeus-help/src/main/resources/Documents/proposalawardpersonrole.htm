Proposal Award Person Role

 

This maintenance document is used to define the various project roles available and used in the KC modules. This document defines which module the role is used in, if the role should follow logic utilized by a Sponsor Hierarchy, if the Unit Details are required, and if the Certification is required.

Editing some aspects of some of the existing roles may be appropriate for your implementation (such as changing Unit Details Required for Key Persons if you wish to implement workflow approval requiremements for key persons), but significant alerations are not advised.
