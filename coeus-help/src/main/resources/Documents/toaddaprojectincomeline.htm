To Add a Project Income line:

1.  Click the [+Add Income] button to open the window

2.  Click in the Period field to select a budget period from the list. Only budget periods defined in this budget version will appear on this list.

3.  Enter a Description of the income in that field.

4.  Enter the reportable dollar amount in the Income field in numerals only.

5.  .Click the [Add] button to save this line entry. Otherwise, click [cancel] to close the window.

6.  Repeat for each budget period or for multiple income types in the same period, as needed.
