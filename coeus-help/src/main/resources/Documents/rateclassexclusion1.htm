Rate Class Exclusion

 

The Rate Class Exclusion maintenance document is used to maintain which rate types  should be excluded from the specified rate class base for calculations. Refer to the Rate Class and  Rate Types documents for the currently maintained types.

Document Layout

Edit or Add a Rate Class Exclusion item:

Click [edit] in the list view to open an existing maintenance document.

Click [Create new] in the upper right corner of the screen to generate a new document.
