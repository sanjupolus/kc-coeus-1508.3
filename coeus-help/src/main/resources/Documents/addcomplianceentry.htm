Add Compliance Entry

Use this manual entry process if you are not using the link to IRB or IACUC parameters:

1.  Click the Add Compliance Entry button

2.  Select a Type from the drop down list (the type selected may change the next options)

3.  Select an Approval Status ( the type selected may change the next options)

4.  Select or search for a Protocol Number, if appropriate.

5.  Enter or select an Application Date by clicking on the calendar tool, if appropriate.

6.  Enter or select an Approval date by clicking on the calendar tool, if appropriate.

7.  Enter or select an Expiration date by clicking on the calendar tool, if appropriate.

8.  Select Exemption # from the drop-down list, if appropriate.

9.  Enter a comment, if appropriate.

10. *If maintained in your implementation, the “create protocol” button may appear. Click this button to have the system generate a new IRB or IACACU protocol document for you to complete and submit for review.

11. Click Add Entry to apply this protocol/review to your proposal, otherwise, click Cancel.
