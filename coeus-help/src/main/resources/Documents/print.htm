Print

Use the Print tool to generate the following files:

Ø Grants.gov form previews, when an s2s opportunity is connected.

Ø Sponsor form packages: generic print forms based on paper-based forms.

Ø Reports:

o  Current Funds Report: generate a list of the proposal investigator’s current awards in the Awards module.

o  Pending Funds Report: generate a list of the proposal investigators pending awards from the Institutional Proposals module.
