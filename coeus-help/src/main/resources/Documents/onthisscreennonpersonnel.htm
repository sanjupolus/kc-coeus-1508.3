You are here:KC Budget 6.0 > Non-Personnel Costs

On this screen – Non-Personnel:

A.  Non-Personnel Costs: Left Navigation button click to open the Non-Personnel Costs screen.

B.  Period Tabs: A tab appears for each project period defined for this proposal, based on the Start and End dates defined in the Proposal Details.

o  Defaults to Period 1 upon initial navigation. While the later periods are presented, users should not select the tabs and manually add personnel. Use the Autocalculate Periods tool to quickly generate all the ‘out-year’ expenses using the maintained inflation rates.

o  Users may select the later periods after they have been Auto-calculated to review and edit. Users may manually add expense to the later periods without using the generate tool, but this method is tedious and time consuming for most budget scenarios.

C.   Assign Non-Personnel button: Click to open the Add Assigned Non-Personnel window.

o  Use the Add window to select the budget category and object code, and enter the cost.

o  A detail rows appear on this screen after the Add process is complete.

D.  Details: Click to open the Details window for the selected item. This window contains three (3) tabs:

o  Details: Provides the ability to edit the start/end dates, budget category, object code, base cost, quantity, description, inflation application, submit cost sharing, on/off campus status, and the apply to later periods  and sync to  cost limit tools, among other items. Refer to the instructions for full details.

o  Cost Sharing: Provides the cost sharing entry field, and a justification text entry box that can be used to help prepare budget notes. Refer to the instructions for full details.

o  Rates:  Displays the calculated overhead rates and the Apply checkbox. Refer to the instructions for full details.

E.   Action Column items:

o  Details: click to open the edit assigned personnel details window (e.g. period type, start and end dates, effort %, charged %)

o  Trashcan icon: click to delete the assigned personnel row from the period.

F.   Proposal Footer actions:

o  Back: click to navigate to the previous screen on the left navigation bar

o  Save: click to save the proposal data and stay on this screen

o  Save and Continue: click to navigate to the next screen listed below on the left navigation bar

o  Complete Budget: click to mark the budget status as complete in the Budget Settings.
