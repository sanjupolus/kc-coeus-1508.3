Proposal Dev Editable Columns

 

Lookup screen label:  Proposal Columns To Alter Lookup

Maintenance doc label:  Proposal Column To Alter

The Proposal Dev Editable Columns maintenance document is used to define which fields in an Enroute proposal should be able to be changed by an administrator using the Data Overide feature. While many columns and fields may appear in the list, changing some items by override may not be appropriate. For example, changing the activity type in override will not recalculate the budget when the F&A rate is different for the new type. Investigate the items you wish to override thoroughly before adding them.

Edit ProposalColumnsToAlter Section

The Edit ProposalColumnsToAlter section of the Proposal Dev Editable Columns maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 738 Proposal Dev Editable Columns Maintenance Document - Edit ProposalColumnsToAlter Section Field Descriptions

Field (Column)

Description

Column Name

Required.  The unique identifier of the proposal column.

Column Label

Required.  The text that will serve as the user interface label next to the column.

Data Length

Enter a numeric value for the maximum amount of characters allowed to be entered.

Data Type

Required.  Click the down arrow  to display the list and click on an item in the list to highlight and select it to populate the box with your selection.  String, Date, Number.

Has Lookup

Required.  Click within the checkbox  to select the item.  A green check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

Lookup Argument

Select one of the following options from the list:  Activity Type, Description, or Higher education function code.  Click the down arrow  to display the list and click on an item in the list to highlight and select it to populate the box with your selection.

Lookup Return

Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection (for example, Description).
