Manually Enter a Summary Budget

There are situations where a detailed budget is not required, but Direct, Indirect (F&A), and Total budget costs must be available for sponsor forms or institutional submissions. The KC Periods & Totals screen allows the user to manually enter values in the Direct and Indirect fields. These values will be mapped to the S2S cover page form (SF 424 series), and populate the KC Institutional Proposal.

o  Click the Edit button in the Action column at the end of the period row you want to maintain: the fields in the table will update to an editable state.

o  Type in the dollar values for the Direct Cost, F&A Cost, Unrecovered F&A (if appropriate), and Cost Sharing (if appropriate).

Ø The system will populate the sum of the Direct and F&A to the Total Sponsor Cost field upon save.

o  Click the Save button in the Action column at the end of the row, or click Cancel to close without saving the changes.

o  Repeat the edit process for each budget period

o  Click the Save button in the footer.
