Custom Report Details


This maintenance document supports defining custom report, including the name, type, permission required to run the report. You must first create the Custom Report Type, and Custom Report Document.

 

 

Use the search feature on the maintenance document to locate existing reports to review or edit.

Use the [create new] button to add a new custome report.

 

Report Label:

Required field. Enter the name of the report.

Report Description:

Enter a description for this report

Report Type Code:

Enter or search for an existing Report Type. See the Report Type maintence document for details.

Permission Name:

Enter the KC Permission required to access this report. (See System Admin > Permissions to review the list of report-based permissions)

Report Design:

upload or replace the design file for this custom report.
